# ⚓ ヨットダイスゲーム

最大4人でオンライン対戦できるブラウザベースのヨット（ダイス）ゲームです。  
Supabaseのリアルタイムデータベースを使って対戦データを同期します。

---

## 🎮 ゲームルール

- サイコロ5個を最大3回振る
- 12カテゴリにスコアを記録していく
- 12ラウンド後、合計スコアが最も高いプレイヤーが勝利

### スコアカテゴリ

| カテゴリ | 説明 |
|---------|------|
| ワンズ〜シックスズ | それぞれの目の合計点 |
| ボーナス | 上半部合計が63点以上で +35点 |
| チャンス | 全サイコロの合計 |
| フルハウス | 3つ+2つ同じ目 → **25点** |
| フォーオブアカインド | 4つ以上同じ目 → **40点** |
| スモールストレート | 4連続 → **30点** |
| ラージストレート | 5連続 → **40点** |
| ヨット！ | 5つ全部同じ目 → **50点** |

## 🚀 セットアップ手順

### 1. Supabaseアカウント作成

1. [https://supabase.com](https://supabase.com) にアクセス
2. 無料アカウントを作成してプロジェクトを新規作成

### 2. データベースのセットアップ

1. Supabaseダッシュボードを開く
2. 左メニューの **SQL Editor** をクリック
3. `supabase-setup.sql` の内容をコピーして貼り付け
4. **Run** ボタンを押して実行
5. `ヨットゲーム セットアップ完了！` というメッセージが出ればOK

### 3. APIキーの取得

1. Supabaseダッシュボードの左メニュー **Settings** → **API** を開く
2. 以下の2つをメモする：
   - **Project URL** (例: `https://xxxxxxxxxx.supabase.co`)
   - **anon/public key** (長い文字列)

### 4. ゲームの起動

1. `index.html` をブラウザで開く
2. Project URLとAnon Keyを入力して「接続する」をクリック
3. 名前を入力してルームを作成/参加！

## 🎲 遊び方

### ルームを作成する場合
1. 「ルームを作成」をクリック
2. 表示される**4文字のルームコード**を友達に共有
3. 2〜4人が揃ったらホストが「ゲーム開始！」をクリック

### ルームに参加する場合
1. 友達から教えてもらったコードを「ルームコード」欄に入力
2. 「参加する」をクリック

### ゲーム中の操作
1. 自分のターンになったら「🎲 振る！」ボタンをクリック
2. キープしたいサイコロをクリックしてホールド（金色にハイライト）
3. 最大3回振れる
4. スコアシートのカテゴリをクリックしてスコアを記録
